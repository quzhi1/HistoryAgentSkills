# 正史引用指南

## 基本原则：所有史料一律给出处

**凡涉及史料的回答，必须标明出处，且出处必须包含：**

1. **书名**（如《魏书》《旧唐书》）
2. **章节名**（卷数 + 传/纪/志/表名，如卷三五《崔浩传》、卷一八三《罗士信传》）

**禁止**：只写「据史书记载」、只写书名不写卷章、或只写内容不写出处。无书名、章节名的引用视为不合格。

---

## 为什么要引用正史？

1. **权威性**：正史是官方认可的历史记录
2. **原始性**：最接近历史事件的一手资料
3. **可验证性**：有明确的卷章出处，便于查证
4. **学术规范**：严肃的历史研究必须引用原始史料

## 当前系统的查询能力

### ✅ 可以做到

1. **通过辞典了解史料出处**
   - 《中国历史大辞典》会提及原始史料来源
   - 例如："见《旧唐书》卷XXX"

2. **推断正史来源**
   - 根据人物朝代判断应查哪部正史
   - 根据辞典内容推断史书卷章

3. **标注原始出处**
   - 在回答中明确标明"据《XX史》记载"
   - 注明具体的卷数和传记名称

### ⚠️ 当前限制

1. **无法直接获取正史原文**
   - 古籍API主要返回HTML页面，不是结构化JSON
   - 需要人工到相应史书中查找原文

2. **依赖辞典整理**
   - 目前引用的是辞典整理后的内容
   - 但会标明原始出处，方便进一步查证

## 标准引用格式（必须包含书名+章节名）

### 格式1：引用正史（理想状态）

**出处须写全：书名 + 卷数 + 传/纪/志名。**

```markdown
据《[正史名]》卷XX《[传记/纪名]》记载：

「[史书原文]」

**出处**：《[正史名]》卷XX《[传记名]》
译文：[现代汉语翻译]
```

**示例**：
```markdown
据《旧唐书》卷183《罗士信传》记载：

「罗士信，齐州历城人也。年十四，从通守张须陀击贼...」

译文：罗士信，齐州历城人。十四岁时，跟随通守张须陀攻打贼寇...
```

### 格式2：基于辞典标注出处（当前可行）

**原始史料出处必须写清书名与章节名，不得省略。**

```markdown
根据《中国历史大辞典》，[人名/事件]的相关记载为：

「[辞典内容]」

**原始史料出处**（书名+章节名）：《[正史名]》卷XX《[传记名]》

（注：以上内容根据辞典整理，详细原文请查阅正史）
```

**示例**：
```markdown
根据《中国历史大辞典》，罗士信的生平事迹为：

「罗士信...唐齐州历城人。隋大业中，从张须陀击王薄、卢明月，年仅十四，作战勇猛...」

**原始史料出处**：《旧唐书》卷183《罗士信传》、《新唐书》卷192《罗士信传》

（注：以上内容根据辞典整理，详细原文请查阅正史）
```

## 引用正史的注意事项

### 1. 明确卷数和传记名（强制：所有史料一律给出处）

**出处必须包含书名与章节名，缺一不可。**

❌ **不好**：据《旧唐书》记载...（无卷、传名）
✅ **好**：据《旧唐书》卷183《罗士信传》记载...

### 2. 区分"新""旧"

- 《旧唐书》vs《新唐书》
- 《旧五代史》vs《新五代史》

两者内容有差异，应明确引用哪一部。

### 3. 引用原文用古汉语

✅ **正确**：「罗士信，齐州历城人也」
❌ **错误**：罗士信是齐州历城人

原文应保持古文形式，然后另外提供现代汉语译文。

### 4. 标注繁简体

正史原文通常是繁体，引用时：
- 保持原文的繁体字
- 或说明"以下引文已转为简体"

## 实例演示

### 问题：李白是哪个朝代的诗人？

#### 当前回答方式

```markdown
根据《中国历史大辞典》：
「李白（701-762），字太白，号青莲居士。唐代伟大诗人...」

**原始史料出处**：
- 《旧唐书》卷190下《李白传》
- 《新唐书》卷202《李白传》

（注：以上内容根据辞典整理，原始传记详见正史）
```

#### 理想回答方式（未来）

```markdown
据《旧唐书》卷190下《李白传》记载：

「李白，字太白，兴圣皇帝九世孙。其先隋末以罪徙西域，神龙初，遁还，客巴西。白之生，母梦长庚星，因以命之...」

译文：李白，字太白，是兴圣皇帝（李暠）的九世孙。他的祖先在隋末因罪被流放西域，神龙初年逃回，客居巴西。李白出生时，其母梦见长庚星（太白金星），因此为他取名为"白"...

据《新唐书》卷202《李白传》记载：

「李白，字太白，兴圣皇帝九世孙。其先隋末以罪徙西域，神龙初，遁还，客巴西...」

（内容与《旧唐书》大致相同，但《新唐书》记载更为简练）
```
